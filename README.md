This is my OS concepts repo, where I document and practice key operating system fundamentals.  
It includes notes, examples, and code snippets on topics like:

- 🔹 CPU Scheduling  
- 🔹 Memory Management  
- 🔹 File Systems  
- 🔹 Synchronization & Deadlocks

- To serve as a study resource for revision and exam prep.

- To share knowledge with peers and collaborators.

- Browse the folders for topic-wise notes and code.  
- Run the sample programs to understand concepts in action.  
- Use this repo as a quick reference while studying.
